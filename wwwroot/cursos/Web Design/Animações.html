Animações

Animações em CSS permitem definir uma quantidade específica de quadros e o que acontecerá entre eles. As animações são configuradas da mesma maneira que as transições. A maneira básica de se criar uma animação é a seguinte:

<div>Animações em ação!</div>
<style>
@keyframes minhaAnimacao {
    from { background-color: green; }
    to { background-color: red; }
}

No exemplo acima, utilizamos a palavra-chave @keyframes para definir os quadros-chave de nossa animação, nesse caso apenas dois, indicando o estado inicial do elemento e o estado final. Definimos o identificador da animação como minhaAnimacao. Ainda dentro do mesmo CSS podemos aplicá-la a um elemento e também configurá-la:

div {
    animation-name: minhaAnimacao;
    animation-duration: 2s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    padding: 50px;
    background-color: black;
}
</style>

Neste exemplo, indicamos o nome da nossa animação com a propriedade animation-name, a duração da animação com a propriedade animation-duration, a quantidade de repetições com a propriedade animation-iteration-count, nesse caso, infinita, e a direção da animação com a propriedade animation-direction, que nesse caso faz a animação se repetir de trás para frente ao chegar no fim.

Keyframes

Como se pode notar a animação é criada fora de um seletor, como um bloco de código independente que pode ser utilizado várias vezes, por exemplo:

@keyframes minhaAnimacao {
    from { left: 0; } to { left: 50px; }
}


p:hover { animation-name: minhaAnimacao; }


div:hover { animation-name: minhaAnimacao; }


button:active { animation-name: minhaAnimacao; }

O exemplo acima é apenas ilustrativo, mas vamos, a partir de agora estudar um pouco mais como funciona a estrutura de uma animação. Antes de tudo, podemos animar diversas propriedades CSS: 

@keyframes minhaAnimacao {
    from {
        background-color: red;
        font-size: 50pt;
        color: yellow;
    }
    to {
        background-color: green;
        font-size: 75pt;
        color: white;
    }
}

E podemos definir quantos quadros chaves forem necessários indicando a porcentagem de tempo que eles demoram para iniciar:

@keyframes minhaAnimacao {
    0% { background-color: white;  }
    25% { background-color: cyan;  }
    50% { background-color: blue;  }
    75% { background-color: cyan;  }
    100% { background-color: white;  }
}

Vários quadros podem ser declarados para um mesmo estado separando-os por vírgula, dessa maneira, podemo simplificar o exemplo anterior:

@keyframes minhaAnimacao {
    0%, 100% { background-color: white;  }
    25%, 75% { background-color: cyan;  }
    50% { background-color: blue;  }
}

Propriedades da animação

Feito esta introdução aos quadros-chave, podemos finalmente utilizar nossa animação quantas vezes necessário e configurá-la da maneira mais adequada a cada situação.

Para se utilizar uma animação é necessário declarar pelo menos duas propriedades que são animation-name e animation-duration, sem elas, a animação não terá efeito algum.

div:hover {
    animation-name: minhaAnimacao;
    animation-duration: 4s;
}

Para analisar as propriedades básicas disponíveis, considere o seguinte código:

<button>Alerta</button>
<style>
button {
    width: 300px;
    height: 100px;
    background-color: yellow;
    border: 10px solid darkred;
    border-radius: 50px;
    margin: auto;
    margin-top: 50px;
    display: block;
    color: black;
    font: bolder 30pt sans-serif;
}
</style>

Criamos um botão e o estilizamos para que fique da seguinte maneira:

Vamos criar dentro da tag de estilo, uma animação que faça nosso botão ficar piscando:

@keyframes alerta {
    0%, 5%, 95%, 100% {
        box-shadow: 0px 0px 0px 25px darkred;
    }
    35%, 65% {
        box-shadow: 0px 0px 0px 5px #F002;
    }
}

A animação é dividida em três etapas, uma de acréscimo, uma pausa no meio e uma de decréscimo, dá a sensação de batida de coração ao efeito de alerta. Vamos então experimentar as propriedades para configurar nossa animação:

button {
    animation-name: alerta;
    animation-duration: 1s;
    animation-iteration-count: infinite;
}

Aplicando a animação diretamente ao elemento, sem nenhum seletor de pseudo-classe, fazemos com que o botão seja animado automaticamente ao renderizar a página. Acima, configuramos a animação para durar um segundo e repetir-se infinitamente.

Adicione a seguinte linha ao final do código anterior:

animation-delay: 5s;

Atualize a página e verá o efeito de delay em ação, a animação demorará cinco segundos para iniciar.

Podemos definir o estado da animação como em execução:

animation-play-state: running;

Ou pausada:

animation-play-state: paused;

Essas propriedades são úteis principalmente quando utilizamos Javascript para controlar a animação.

Em vez de permitir a animação repetir-se infinitamente, podemos definir a quantidade de loops que ocorrerá:

animation-iteration-count: 3;

Neste caso, você deve substituir a linha do CSS que definiu a repetição como infinite.

Podemos fazer com que a animação corra de trás para frente:

animation-direction: reverse;

Ou que alterne, inicio-fim-fim-inicio… :

animation-direction: alternate;

E também que alterne, mas iniciando do final, fim-inicio-inicio-fim… :

animation-direction: alternate-reverse;

Curvas de animação

Podemos definir curvas de animação utilizando a propriedade animation-timing-function, e funciona exatamente como nas transições, vejamos dois exemplos:

Ease In Out:

animation-timing-function: ease-in-out;

Efeito de chicote Ease In Out:

animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);

Animações e transformações

Transformações em animações visam controlar elementos como mudar de tamanho, fazê-los girar etc., utilize o mesmo HTML e CSS anterior mas substitua a animação pela seguinte:

@keyframes alerta {
    0%, 5%, 95%, 100% {
        transform: rotate(0deg);
    }
    35%, 65% {
        transform: rotate(180deg) scale(150%);
    }
}

Estamos aplicando à nossa animação um efeito de rotação em 180 graus e de ampliação em 150%, vamos aplicá-la da seguinte maneira:

button {
    animation-name: alerta;
    animation-duration: 3s;
    animation-iteration-count: infinite;
    animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

A duração será de três segundos, terá um efeito de chicote e se repetirá infinitamente.
