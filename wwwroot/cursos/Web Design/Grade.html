Grade

Um layout de grade é uma maneira de organizar itens com CSS utilizando linhas e colunas de forma prática e segura. Se você não possui familiaridade com este tipo de layout, basta imaginá-lo como as linhas, colunas e células do Excel, sendo uma célula a interseção entre uma linha e coluna. Treine bastante o conceito de grade pois ele é fundamental no layout de páginas web e você o utilizará bastante como profissional.

Grade básica

Uma grade básica pode ser criada com apenas duas propriedades aplicadas ao contêiner, que são o display e um grid-template-column, ou grid-template-row, a grade é criada atribuindo ao contêiner um display:grid e as colunas ou linhas são criadas com uma das dua últimas alternativas, vejamos um exemplo:

<div class="container">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
  <div>Item 5...</div>
</div>

Com o contêiner criado e seus conteúdos devidamente inseridos, basta aplicar um estilo ao contêiner e outro para as divs do conteúdo:

<style>
.container {
  display: grid;
  grid-template-columns: 25% 25% 50%;
}
.container div {
  padding: 10px;
  background-color: red;
}
</style>

Perceba que duas colunas ocuparão metade da grade, e a terceira coluna, a outra metade, além do fato de que a grade é responsiva, pois estamos trabalhando com porcentagem, o que faz com que as colunas se redimensionem de acordo com o tamanho da tela. Isso é suficiente para o que desejamos fazer. Note que mesmo uma div possuindo um display:block por padrão, que não permite que sejam exibidas na mesma linha, elas podem ocupar uma mesma linha da grade sem alterarmos seu display individual. Isso ocorre porque cada espaço criado por uma grade funciona semelhante a mais um contêiner.

Para trabalharmos com o espaçamento entre as células de nossa grade podemos utilizar duas práticas, uma é a margem do conteúdo, definindo a propriedade margin diretamente nos elementos filhos da grade, mas teremos um resultado melhor utilizando o gap da grade, que pode ser definido nas colunas ou nas linhas:

.container {
  display: grid;
  grid-template-columns: 25% 25%;
  column-gap: 5px;
  row-gap: 5px;
}

Acima aplicamos um espaçamento tanto entre as linhas quanto entre as colunas, isto pode ser feito de forma individual, mas se desejamos aplicar ambos, podemos simplificar assim:

.container {
  display: grid;
  grid-template-columns: 25% 25%;
  gap: 5px;
}

A grade produz novas linhas automaticamente, podemos criar linhas com a propriedade grid-template-rows, no entanto se a quantidade de conteúdo exceder a quantidade de linhas predefinidas, novas linhas serão criadas sem o estilo aplicado.

Utilizar porcentagem e margem ou gap faz com que seja necessário definir a porcentagem utilizando a função calc() para subtrair o tamanho do gap ou da margem, isso é facilmente contornado utilizando a unidade de medida fr.

Unidades de medida

Colunas e linhas da grade podem ser definidas em outras medidas, como pixels por exemplo:

.container {
  display: grid;
  grid-template-columns: 200px 200px 150px;
}

No entanto, a unidade de medida mais utilizada é o tamanho flex fr, que facilita a criação e redimensiona as linhas ou colunas automaticamente:

grid-template-columns: 2fr 3fr 1fr 1fr;

Neste exemplo a primeira coluna terá o dobro da largura que a terceira e a quarta, e a segunda coluna terá o triplo. Recomendo que utilize esta unidade de medida ao criar suas grades sempre que possível!

Mais responsividade ainda

Algumas funções podem ser combinadas para deixar a criação de grades mais fácil e torná-las ainda mais responsivas, como as funções repeat() e funções de mínimo e máximo. A função repeat() pode ser utilizada para repetir uma quantidade de linhas ou colunas, da seguinte forma:

grid-template-columns: repeat(3, 1fr);
grid-template-rows: repeat(5, 50px);

Neste exemplo criamos 3 colunas de 1fr e 5 linhas de 50px. Tome cuidado ao utilizar essa função pois se criar mais linhas ou colunas do que há de conteúdo, a grade, apesar de não preenchê-las, criará um overflow aumentando sua dimensão.

Podemos ainda utilizar o valor auto-fill combinado com a função min() para preencher automaticamente a grade:

grid-template-columns: repeat(auto-fill, min(500px));

No exemplo acima, a grade criará quantas colunas possíveis com uma largura mínima de 500px. Isso faz com que ao redimensionar a tela, as colunas diminuam de quantidade para mantê-las no valor mínimo exigido.
