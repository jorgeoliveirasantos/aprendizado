Ids e classes

Os atributos id e class são atributos de identificação utilizados em várias ocasiões, como navegação, seleção de CSS, manipulação com Javascript entre outras:

<p id="txtId" class="txtClasse">Um texto qualquer...</p>

Gerando alguma confusão nas mentes principiantes, estes dois tipos de identificadores possuem profundas diferenças, pois servem a propósitos distintos. As principais diferenças são:

1. Uma id por elemento, várias classes por elemento.

2. Um elemento por id, vários elementos por classe.

Assim como os nomes indicam, id são identificadores únicos, por este motivo, um elemento só pode ter uma id, e vários elementos não podem possuir a mesma id, pois ao acessar um elemento por id, seja no Javascritp, ou no CSS, é esperado que apenas um elemento seja obtido.

Enquanto isso, classes pretendem criar uma categoria de elementos que compartilham valores em comum, desta maneira, um elemento pode possuir duas ou mais classes, que são definidas no mesmo atributo e separadas por espaço, e vários elementos podem compartilhar a mesma classe:

<p id="p1" class="classe1 classe2">Meu parágrafo1</p>
<p id="p2" class="classe2">Meu parágrafo2</p>
<p id="p3" class="classe3 classe1">Meu parágrafo3</p>

Pegando o exemplo acima, podemos aplicar um estilo assim:

<style>
.classe1 { color: red; }
.classe2 { background-color: green; }
.classe3 { border: yellow 1px solid; }


#p1 { font-size: large; }


#p2 { font-size: larger; }


#p3 { font-size: x-large; }


</style>

Veja o resultado:

Cada um dos elementos recebeu seu tamanho de fonte de acordo com sua id, e a cor de fundo, cor da fonte ou borda, de acordo com a classe que lhe foi aplicada.
