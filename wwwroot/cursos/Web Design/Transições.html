Transições

Quando um objeto muda de estado, por exemplo, quando o mouse está sobre ele, quando um checkbox é marcado, quando um elemento se torna ativo, ele pode receber estilos personalizados, como outra cor de background, aumentar ou diminuir de tamanho etc. Mas essas transformações ocorrem de maneira instantânea, por exemplo um botão que possui uma cor de texto padrão, e outra cor aplicada somente à pseudo-classe hover, mudará de cor instantaneamente quando o mouse estiver sobre ele:

button { background-color: white; }
button:hover { background-color: black; }

As transições visam justamente manipular os estados intermediários, no mesmo exemplo do botão, se a cor do estado normal for branca, e a cor do hover for preta, as transições manipularão todas as escalas de cinza entre o branco e o preto. As transições podem manipular diversos tipos de propriedade. Ainda no exemplo do botão, se quisermos fazer com que a transição entre o estado normal e o estado hover dure um segundo, podemos fazer o seguinte:

button {
    background-color: black;
    transition-duration: 1s;
}
button:hover { background-color: white; }

Dessa maneira, a mudança não será automática, mas poderemos ver a gradação das cores durante a transformação.

Duração, atraso e propriedade

As propriedades de transição são:

transition-delay: define quanto tempo a transição demora para ser iniciada.

transition-duration: define quanto tempo a transição dura.

transition-property: define em quais propriedades CSS a transição será aplicada.

transition-timing-function: define a curva da animação. Veremos adiante.

No exemplo a seguir animaremos a transição de um título definindo o tempo um atraso de 150 milissegundos para o início da transição e uma duração de 850 milissegundos, fazendo com que no total, a transição dure um segundo:

Não utiliza transições muito longas em elementos HTML que exigem interação do usuário, isso pode fazer com que a página pareça lenta, diminuindo a acessibilidade e fazendo com que o usuário tenha uma má experiência de navegação. 

<h1>Transições em ação!</h1>


<style>


h1 {


    transition-property: all;


    transition-duration: 850ms;


    transition-delay: 150ms;


    color: yellow;


    background-color: darkslategrey;


    padding: 25px;


    width: 70%;


    margin: auto;


    font-family: 'Courier New', Courier, monospace;


    font-weight: bolder;


    font-size: 50pt;


    text-align: center;


}


h1:hover {


    font-size: 70pt;


    color: white;


    width: 80%;


}


</style>

Vamos alterar algumas linhas no código? Podemos definir quais propriedades serão afetadas pela animação. Você notou que a pseudo-classe hover define três propriedades, font-size, color e width, no código anterior, todas essas propriedades se comportarão de acordo com as propriedades de transição, mas se alterarmos esta linha:

transition-property: all;

Mudando o valor all para color, somente a cor sofrerá a transição:

transition-property: color;

Se tiver acesso ao editor de código tente fazer esta mudança. Neste caso a mudança das outras propriedades ocorrerá de maneira instantânea, porque as propriedades transition-delay e transition-duration não se aplicam a elas.

Podemos ainda alterar para que a transição se aplique somente à largura:

transition-property: width;

E podemos definir em quantas propriedades queremos aplicar a transição separando-as por vírgula:

transition-property: color, width;

Curvas de animação

Curvas de animação definem a velocidade durante a transição, as básicas são:

Linear: A transição ocorre na mesma velocidade do começo ao fim.

Ease: É semelhante ao ease-out. A animação desacelera um pouco perto do final.

Ease In: A animação começa lenta e acelera conforme se aproxima do final.

Ease Out: A animação começa rápida e desacelera conforme se aproxima do final.

Ease In Out: A animação começa lenta, acelera conforma chega na metade do progresso e depois volta a desacelerar.

A curva de uma animação diz respeito ao arco formado por velocidade e progresso no gráfico. Se tiver curiosidade pesquise um pouco sobre “curvas de animação” ou “curvas de bézier na animação”.

Suponha que você tem o seguinte HTML:

<div id="linear">Linear</div>
<div id="ease">Ease</div>
<div id="easein">Ease In</div>
<div id="easeout">Ease Out</div>
<div id="easeinout">Ease In Out</div>

Criamos cinco elementos div e atribuímos uma id a cada um identificando o tipo de transição que criaremos. Primeiro, vamos estilizar as div:

<style>
div {
    color: yellow;
    background-color: darkslategrey;
    padding: 10px;
    width: 10%;
    margin: auto;
    margin-bottom: 10px;
    font-size: 20pt;
    text-align: center;
}

Agora vamos definir o estado hover delas, o que deverá acontecer quando o usuário passar o mouse sobre:

div:hover {  width: 90%; }

Podemos então começar a definir as propriedades da transição.

Linear:

#linear {
    transition-timing-function: linear;
    transition-duration: 1200ms;
}

Ease:

#ease {
    transition-timing-function: ease;
    transition-duration: 1200ms;
}

Ease In:

#easein {
    transition-timing-function: ease-in;
    transition-duration: 1200ms;
}

Ease Out:

#easeout {
    transition-timing-function: ease-out;
    transition-duration: 1200ms;
}

Ease In Out:

#easeinout {
    transition-timing-function: ease-in-out;
    transition-duration: 1200ms;
}
</style>

Efeito de chicote

Efeitos de chicote na transição são criados com a função cubic-bezier, que pode ser personalizada a gosto caso você possua um conhecimento básico sobre manipulação de curvas de bézier em animações, mas podemos utilizar alguns valores “default” que são:

Chicote Ease Out:

transition-timing-function: cubic-bezier(0.175, 0.885, 0.32, 1.275);

Chicote Ease In:

transition-timing-function: cubic-bezier(0.6, -0.28, 0.735, 0.045);

Chicote Ease In Out:

transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);
