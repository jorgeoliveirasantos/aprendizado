Media Query

Media Query, ou Consulta de Mídia, é uma prática utilizada para modificar o site de acordo com o dispositivo, e por um bom tempo foi a única prática utilizada para tornar as páginas responsivas, antes do desenvolvimento de elementos flex, como flexbox e grade, que são responsivas por natureza.

A consulta de mídia utiliza a palavra-chave @media, e é feita na seguinte sintaxe:

@media (max-width: 600px) {
    seletor { propriedade: valor; }
}

O parâmetro max-width nesse caso, informa que todas as propriedades definidas dentro dessa função serão aplicadas somente se a tela tiver menos que o valor indicado (nesse caso 600px).

Alterando a direção da caixa

Veremos um uso muito comum para o media query que é definir estilos diferentes para diferentes tamanhos de tela, para que você compreenda como funciona, imagine que você possui o seguinte HTML:

<div class="conteiner">
  <img src="1.png">
  <img src="2.png">
  <img src="3.png">
  <img src="4.png">
  <img src="5.png">
  <img src="6.png">
  <img src="7.png">
</div>

Nesse arquivo você deseja que, caso o usuário possua uma tela grande como de um desktop, ele possa navegar pelas imagens rolando-as numa galeria horizontal, mas se a tela for pequena, como um tablet ou celular, ele navegue pelas imagens na vertical, rolando para baixo e vendo-as preenchidas na tela.

Para conseguir isso, vamos primeiro configurar as imagens para uma tela grande, portanto, adicione o seguinte CSS:

<style>
.conteiner {
  background-color: black;
  width: fit-content;
  overflow: auto;
  display: flex;
  flex-direction: row;
  align-items: stretch;
}
.conteiner img {
  box-sizing: border-box;
  border: yellow 5px solid;
  margin: 10px;
}
</style>

No exemplo acima, criamos uma caixa flex que exibe seus itens numa única linha com uma barra de rolagem para navegação horizontal. Agora criaremos um estilo CSS que transforme a direção da caixa flex de linha para coluna:

<style>
@media (max-width: 700px) {
  .conteiner {
    flex-direction: column;
    width: 100%;
  }
}
</style>

Note que bastou alterar a propriedade flex-direction para column, você pode testar o efeito redimensionando a janela.

Alterando a largura da coluna

Outra abordagem eficiente é a da grade, que por si só já é responsiva, mas que podemos controlar ainda mais com consulta de mídia. Para conseguir isso criamos uma div para a grade contêiner, e divs internas para as colunas, atribuindo a cada uma sua respectiva classe:

<div class="grade">
  <div class="col">
      Mussum Ipsum, cacilds vidis litro abertis. Mais vale um bebadis conhecidiss, que um alcoolatra anonimis.Si num tem leite então bota uma pinga aí cumpadi!Casamentiss faiz malandris se pirulitá.Paisis, filhis, espiritis santis.
    </div>
    <div class="col">
      Aenean aliquam molestie leo, vitae iaculis nisl.A ordem dos tratores não altera o pão duris.Posuere libero varius. Nullam a nisl ut ante blandit hendrerit. Aenean sit amet nisi.Nullam volutpat risus nec leo commodo, ut interdum diam laoreet. Sed non consequat odio.
    </div>
    <div class="col">
      Si u mundo tá muito paradis? Toma um mé que o mundo vai girarzis!Admodum accumsan disputationi eu sit. Vide electram sadipscing et per.Aenean aliquam molestie leo, vitae iaculis nisl.Manduma pindureta quium dia nois paga.
  </div>
</div>

Feito isso, devemos então configurar os estilos, a grade deve se ajustar ao conteúdo:

<style>
.grade {
  background-color: black;
  width: fit-content;
}
</style>

Em seguida, por possuirmos três colunas, vamos defini-las com 33% de largura, um box-sizing:border-box, e um display:block:

.col {
  width: 33%;
  color: antiquewhite;
  box-sizing: border-box;
  border: yellow 1px solid;
  display: inline-block;
  padding: 10px;
}

Agora, caso a página tenha menos de 900px, as colunas terão 100% de largura e serão exibidas uma após a outra.

@media (max-width: 900px) {
  .col {
    width: 100%;
  }
}

jsfiddle.net/jorgeoliveiraonline/udpLw62n/7/

Note que depois das técnicas mais modernas, não é preciso utilizar media query para produzir layouts responsivos na maioria dos casos, no código anterior por exemplo, poderíamos substituir a consulta de mídia apenas adicionando uma propriedade flex-basis ao elemento de coluna. Entretanto a consulta de mídia permite maior controle sobre o comportamento do layout da página.
