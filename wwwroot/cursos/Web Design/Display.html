Display

A propriedade display define como um objeto é arranjado em relação aos outros ao seu redor sendo:

inline: o objeto é exibido na mesma linha que os demais e a caixa de borda é ignorada.

inline-block: o objeto é exibido na mesma linha que os outros, mas a caixa é respeitada.

block: o objeto é exibido numa linha individual e a caixa é respeitada.

Elementos <div> e <p> possuem um valor padrão de display block, elementos <li> possuem um padrão list-item e elementos <span> possuem um valor padrão de display inline. Por isso podemos definir propriedades para uma palavra com marcas <span> sem que ela provoque uma quebra de linha no parágrafo ou provoque espaçamentos extras.
