Introdução ao CSS

Estilos são pares de propriedade e valor aplicados a uma seleção, neste modelo:

seletor {
    propriedade: "valor";
}

Onde o seletor diz quem receberá o estilo, por exemplo um elemento definido pela tag p, h1, div… A propriedade diz o que será alterado, por exemplo o background, a cor do conteúdo, o espaçamento. E o valor diz quanto ou como será alterado.

O que é uma Cascata?

A cascata é a hierarquia da aplicação de estilo aos elementos, sendo que se possuirmos um elemento:

<div style="color: black; font-size: 20pt;">
    <p style="color: yellow;">Meu parágrafo</p>
</div>

O elemento parágrafo herdará ambas propriedades de estilo de seu elemento pai <div>, devido o efeito de cascata, no entanto sua própria propriedade color substituirá a propriedade herdada.

Um estilo não é simplesmente substituído, o CSS possui um cálculo para definir a importância e decidir qual estilo será aplicado, veremos isso adiante, mas na maioria dos casos, como o anterior, considerar essa substituição será suficiente.

Onde usar o CSS

Como este assunto já foi abordado, vamos apenas revisar de maneira rápida onde inserir estilos. Espero que tenha acompanhado todo o curso até agora com atenção.

Podemos utilizar CSS em linha:

<p style="color: yellow;">Meu parágrafo</p>

Podemos inserir um elemento de CSS nas tags <head> do arquivo ou ao final do arquivo:

<style>
div {
    background-color: black;
}
</style>

Ou podemos criar um arquivo CSS separado do arquivo do documento HTML e apenas referenciá-lo no documento com um link:

<link rel="stylesheet" href="meuestilo.css">

Neste exemplo, supõe-se que na mesma pasta do documento HTML, haja um arquivo nomeado como meuestilo.css.

Estilos não aplicados

Quando omitimos um estilo CSS, ao renderizar a página, diferentes navegadores aplicarão uma folha de estilo padrão, estas são as primeiras propriedades que os elementos receberão, para se ter uma ideia, observe esta imagem:

Aqui temos um estilo padrão sendo aplicado a um controle progress por diferentes navegadores, este estilo muda também de acordo com a versão do navegador, isso porque os desenvolvedores também efetuam alterações na folha de estilo padrão do navegador. Este estilo é substituído quando começamos a definir suas propriedades explicitamente.
