Contêineres genéricos

Os elementos <div> e <span> pertencem a uma categoria de elementos chamados de “Contêineres Genéricos” que por si só não produzem nenhum resultado, mas que tem um potencial poderoso, por exemplo, todos os sites da web são construídos com divs, pode conferir nas opções de desenvolvedor de seu navegador em qualquer página. Mas se você inserir em seu HTML:

<div></div>

Nada acontecerá, e se inserir:

<div>Meu texto</div>

O resultado será exatamente o mesmo que inserir o texto sem nenhuma tag. No entanto, utilizando atributos para personalizar uma div, você pode ter a sensação de desenhar no documento:

<div style="width: 200px;
   height: 300px;
   display: inline-block;
   margin: 30px;
   background-color: red;"></div>
<div style="width: 2px;
   height: 300px;
   display: inline-block;
   margin: 30px;
   background-color: yellow;"></div>
<div style="width: 200px;
   height: 300px;
   display: inline-block;
   margin: 30px;
   background-color: darkcyan;"></div>

Veja o resultado:

Se você levar em consideração que qualquer elemento herda todos os atributos herdáveis de seu elemento pai, você poderá entender porque todos os sites são construídos com divs, veja este exemplo:

<div style="color: red;">
  <p>Parágrafo 1</p>
  <p>Parágrafo 2</p>
</div>
<div style="color: green;">
  <p>Parágrafo 1</p>
  <p>Parágrafo 2</p>
</div>
<div style="color: blue;">
  <p>Parágrafo 1</p>
  <p>Parágrafo 2</p>
</div>

Porque usar div

Uma div pode ainda conter outras div dentro de si, que contenham outras que contenham outras, indefinidamente. Isso permite estilizar o documento de maneira muito mais prática que sair estilizando elemento por elemento. Um cabeçalho, por exemplo, pode conter um título, um subtítulo, uma descrição e até mesmo uma imagem. Mas se precisar em algum momento ocultar o cabeçalho, você teria que sair ocultando um por um, e pode ser que estes não sejam os únicos elementos do cabeçalho, a imagem, por exemplo, pode ter uma legenda. Nesse caso, todos os elementos seriam colocados dentro de uma div, que teria um id ou uma classe, nesse caso, bastaria alterar a propriedade da div, e não de todos os elementos.

É fácil compreender uma div pensando nela como um grupo de elementos! Você pode manipular um grupo como manipularia outros elementos.

Enquanto a div é utilizada para agrupar elementos, o span é utilizado para separar, na realidade, para selecionar, uma parte do conteúdo.

O span possui um display inline por padrão e a div possui um display block. O conteúdo entre uma tag span permanecem na mesma linha e têm suas margens ignoradas enquanto elementos div ocupam sua própria linha e têm suas margens respeitadas. 

<html>
<p>
  Mussum Ipsum, cacilds vidis litro abertis.
  Atirei o pau no gatis, per gatis num morreus.
  <span id="ressaltado">Praesent vel viverra nisi.</span>
  Mauris aliquet nunc non turpis scelerisque, eget.
  Quem manda na minha terra sou euzis!
  Delegadis gente finis, bibendum egestas augue arcu ut est.
</p>
</html>
<style>
#ressaltado {
  font-weight: bold;
  color: aqua;
}
</style>

Neste exemplo, todo o conteúdo entre as tags span terão seu estilo alterado.

É possível alterar o estilo do span para que ele tenha um display block, e uma div para que tenha display inline ou inline-block. Estudaremos isso no momento certo.

Podemos utilizar um span como desejarmos através de atributos, um exemplo legal para aprendermos é desenhar um círculo, veja um exemplo:

<span id="circulo"></span>
<style>
#circulo {
  background-color: black;
  height: 15px;
  width: 15px;
  border-radius: 50%;
}
</style>

Lembrando que este círculo pode ser exibido no meio de uma linha de texto, ou num botão por exemplo.
