Tags personalizadas

Na realidade, elementos HTML podem ser qualquer coisa, por exemplo:

<personalizado>
  Texto do elemento personalizado
</personalizado>
<style>
personalizado {
  color: black;
}
</style>

Neste exemplo criamos nossa própria tag chamada <personalizado> e inserimos dentro dela um texto e utilizamos o nome da tag como seletor do CSS para aplicar uma cor de texto. Você já sabe que textos podem ser inseridos até mesmo fora de qualquer tag que serão interpretados pelo navegador, mas não terão nenhuma maneira de estilização, as tags permitem a seleção de todo o conteúdo dentro delas, herdam o estilo CSS padrão que o navegador aplica aos elementos que não forma personalizados e herdam também os atributos globais, podemos utilizar esses atributos globais em nossa tag personalizada:

<personalizado title="Minha tag personalizada">
  Texto do elemento personalizado
</personalizado>

Agora definimos uma propriedade title a ser exibida quando o usuário passar o mouse sobre o elemento. Elementos personalizados podem também manipular eventos, como o evento de clique:

<personalizado onclick="document.location = 'minhaPagina2.html'">
  Texto do elemento personalizado
</personalizado>

Agora ao clicar no elemento personalizado seremos enviados à outra página HTML. Vários outros atributos globais podem ser utilizados.

Tags HTML personalizadas podem ser utilizadas em vários documentos com o mesmo nome, mas estarão no escopo apenas do documento ao qual pertencem. Podemos criar Elementos Reutilizáveis por meio de Javascript, que podem conter conteúdo a ser exibido em vários documentos HTML, como menus, barras laterais etc. Veremos isso mais adiante.
