Posicionamento em grades

Após esta breve introdução, para te acostumar com a técnica, vamos começar a criar grades de verdade e da melhor forma. As áreas em grades permitem nomear nossos elementos e aplicá-los à grade decidindo onde estarão, como serão configurados, quantos espaços ocuparão etc.

Imagine que você possui o seguinte HTML:

<div class="conteiner">
  <header>
    <h1>Minha página pessoal</h1>
    <p>Êta página porreta!</p>
  </header>
      <aside>
        <h2>Barra lateral</h2>
        <p>Mussum Ipsum, cacilds vidis litro abertis. Tá deprimidis, eu conheço uma cachacis que pode alegrar sua vidis.Si num tem leite então bota uma pinga aí cumpadi!Suco de cevadiss, é um leite divinis, qui tem lupuliz, matis, aguis e fermentis.Diuretics paradis num copo é motivis de denguis.</p>
      </aside>
      
      <div class="conteudo">
        <h2>Meu título</h2>
        <p>Nullam volutpat risus nec leo commodo, ut interdum diam laoreet. Sed non consequat odio.Tá deprimidis, eu conheço uma cachacis que pode alegrar sua vidis.Não sou faixa preta cumpadi, sou preto inteiris, inteiris.Posuere libero varius. Nullam a nisl ut ante blandit hendrerit. Aenean sit amet nisi.</p>
  </div>


  <footer>


    <p>Este é meu rodapé.</p>


    <p>meusite@site.it</p>


  </footer>


</div>

Aqui temos um site completo, com cabeçalho, barra lateral, conteúdo e rodapé. Podemos configurar nosso site de maneira muito prática com uma grade. Antes de tudo, o que devemos fazer é estilizar o site, o que faremos da seguinte maneira:

<style>
  body {
    width: 80%;
    margin: 20px auto auto auto;
    font-family: sans-serif;
    background-color: #222;
    color: white;
  }
  header, aside, footer {
    padding: 10px;
    border: 1px solid goldenrod;
  }
</style>

Definimos as cores e o dimensionamento dos itens, começaremos então a configurar a grade, primeiro criando-a no contêiner:

<style>
  .conteiner {
    display: grid;
    grid-template-columns: 1fr 3fr;
    gap: 20px;
  }
</style>

Veja que criamos uma grade com apenas duas colunas, isso porque a primeira com um quarto do tamanho será ocupada pela barra lateral (elemento aside), a segunda será ocupada pelo conteúdo do site (elemento div). O cabeçalho e o rodapé devem ocupar toda a largura da página.

Para posicionar os elementos conforme descrito, faremos o seguinte:

<style>
  header {
    background-color: black;
    grid-column-start: 1;
    grid-column-end: 3;
    grid-row: 1;
    text-align: center;
  }
  footer {
    background-color: black;
    grid-column-start: 1;
    grid-column-end: 3;
    grid-row: 3;
    text-align: center;
  }
  .conteudo {
    grid-column: 2;
  }
  aside {
    grid-column: 1;
  }
</style>

Enquanto as propriedades grid-row e grid-column definiram o posicionamento dos elementos, as propriedades, grid-column-start e grid-column-end definiram o span horizontal, isto é quantas colunas o elemento deve ocupar.

grid-column: define a coluna a ser ocupada ou seu span horizontal.

grid-row: define a linha a ser ocupada ou seu span vertical.

grid-column-start: define o início do span horizontal.

grid-column-end: define o fim do span horizontal (última coluna ocupada +1).

grid-row-start: define o início do span vertical.

grid-row-end: define o fim do span vertical (última linha ocupada +1).

Podemos definir o span de colunas ou de linhas passando a coluna inicial e final, ou a linha inicial e final, com as propriedades grid-column, e grid-row:

grid-column: 1/3
grid-row: 1/5

Na primeira linha do código acima, o elemento ocupará a primeira e a segunda colunas, e na segunda linha o elemento terá um span vertical, ocupando da primeira à quarta linha da grade.

Quer facilitar ainda mais? Use áreas:

.conteiner {
  display: grid;
  grid-template-columns: 1fr 3fr;
  grid-template-areas:
    "cabecalho cabecalho"
    "lateral conteudo"
    "rodape rodape";
  gap: 20px;
}
header {  grid-area: cabecalho;  }
footer {  grid-area: rodape;  }
.conteudo {  grid-area: conteudo;  }
aside {  grid-area: lateral;  }

Veja como é prático isso! Cada linha da propriedade grid-template-areas cria uma linha da grade e cada valor na linha cria uma coluna (na realidade cada declaração entre aspas, mas para facilitar utilizamos quebras de linha para separar as declarações). Criamos as linhas da grade apenas com linhas de texto, na primeira linha da propriedade grid-template-areas criamos uma área que ocupa duas colunas, e atribuímos essa área (cabecalho) ao elemento header, utilizamos a mesma técnica com os outros elementos.
