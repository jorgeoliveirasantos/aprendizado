Fontes

A formatação de texto no CSS conta com uma grande quantidade de recursos para termos toda controle na aparência da tipografia para o usuário, incluindo alinhamento, espaçamentos, estilos etc. A formatação básica é feita definindo a fonte, a cor do texto, o estilo do caractere e o tamanho.

Para definir o tamanho da fonte, basta definir a propriedade font-size:

p { font-size: 36pt; }

O valor padrão da propriedade font-size é medium ou o valor herdado. Os valores predefinidos são, na ordem crescente (do menor ao maior):

xx-small, x-small, smaller, small, medium, large, larger,

x-large, xx-large

No entanto, o tamanho da fonte pode ser ainda definido em pontos, assim:

font-size: 36pt;

Ou em pixels:

font-size: 100px;

O valor em pixels corresponde à altura total do texto contando as hastes inferiores e superiores e não somente o corpo do texto.

Podemos ainda definir o tamanho de acordo com um tamanho herdado, por exemplo:

font-size: 2em;

No exemplo acima, o texto terá o dobro do tamanho herdado. Lembrando que parágrafos e títulos herdam tamanhos diferentes, e ainda que caso o texto não herde um tamanho definido explicitamente em algum momento, herdará o tamanho padrão definido pelo navegador. Para entender como funciona considere o seguinte:

<div>
    <p>Fonte 12pt</p>
    <p id="p1">Fonte 2em</p>
</div>
<style>
div { font-size: 12pt; }
#p1 { font-size: 2em; }
</style>

Neste exemplo, temos um elemento div contendo dois parágrafos em seu interior, o primeiro estilo CSS aplica à div um tamanho de fonte 12pt, este estilo será herdado por todos os parágrafos contidos dentro dela, como é o caso do primeiro parágrafo.

O tamanho 2em definido no segundo parágrafo na realidade diz que o tamanho do texto será 2 vezes o valor herdado, nesse caso, da div.

Você pode ainda definir o tamanho com a unidade de medida rem, em vez de em, nesse caso, 1.5rem será 1.5 vezes o tamanho definido pelo navegador e não o valor herdado, ou seja, essa medida se baseia no valor raiz dos elementos, independente de que valor ele herda de outros estilos.

Estilo de caractere

Podemos definir estilos negrito, itálico, sublinhado, além de espaçamento, entre outros estilos de caractere, vejamos alguns exemplos:

Itálico:

font-style: italic;

Sublinhado:

text-decoration: underline;

Tachado:

text-decoration: line-through;

Versalete:

font-variant: small-caps;

Em relação ao negrito, ele pode ser definido em gradação, podendo ser predefinida, indo do lighter, ou fio de cabelo, até o bolder, que é o negrito mais espesso:

font-weight: lighter;
font-weight: normal;
font-weight: bold;
font-weight: bolder;

Mas pode também ser definido um “peso” para a fonte com valores múltiplos de 100 que vão de 100 a 900, sendo 100 o lighter, e 900 o bolder:

font-weight: 300;

Relembrando que essa variação precisa ser suportada pela fonte utilizada, por exemplo, se a fonte não suporta lighter, definir o peso 100 a tornará normal, se ela não suporta bolder, definir o peso 900 a tornará bold.

Podemos ainda alterar a caixa da fonte.

Para iniciais maiúsculas:

text-transform: capitalize;

Para minúsculas:

text-transform: lowercase;

Para maiúsculas:

text-transform: uppercase;

Para controlar o espaçamento da fonte, temos duas opções, a primeira é a propriedade letter-spacing:

letter-spacing: 1px;
letter-spacing: 2pt;

E a segunda é a propriedade font-stretch que possui valores predefinidos:

font-stretch: condensed;

Os valores disponíveis para essa propriedade são, na ordem do mais condensado até o mais espaçado:

ultra-condensed, condensed, semi-condensed, normal,

semi-expanded, expanded, ultra-expanded.

Os valores de font-stretch, se você é acostumado com tipografia, deve perceber que são os mesmos que acompanham as famílias de fonte, portanto essas opções só surtirão efeito se estiverem disponíveis na fonte. Na dúvida utilize letter-spacing.

Famílias de fontes

Em relação às famílias de fontes temos várias opções, podemos desde definir fontes personalizadas, até importar fontes da internet, mas comecemos pelo básico, definindo o estilo de “tipo”, que pode ser:

Sem serifa (utilizada nos títulos deste livro):

font-family: sans-serif;

Serifada (para corpo do texto, usada neste livro):

font-family: serif;

Mono-espaçada (código):

font-family: monospace;

Cursiva (à mão):

font-family: cursive;

Garamond (utilizada neste livro):

font-family: garamond;

Na mesma propriedade font-family podemos definir a fonte principal do nosso texto, assim:

font-family: Arial;
font-family: Garamond;
font-family: 'Times New Roman';

Para que o usuário receba o texto na fonte que você definiu, ela deve estar disponível pelo navegador, caso contrário a fonte padrão do navegador será utilizada.

O alerta acima chama a atenção para algo muito importante que é a disponibilidade da fonte, podemos contornar este problema de duas maneiras, a primeira é definindo uma cascata de fontes alternativas (fallback), a segunda é definindo uma fonte on-line ou do arquivo.

As fontes alternativas são definidas em sequência, se a primeira não estiver disponível o navegador passa para a próxima, e assim por diante, somente se todas estiverem indisponíveis que a fonte padrão do navegador será usada, veja um exemplo:

font-family: Arial, Helvetica, sans-serif;

Neste exemplo a primeira fonte é Arial, se não estiver disponível, o navegador tentará Helvética, caso não consiga, utilizará a fonte sem serifa padrão.

Ao estilizar seus textos é importante definir como última opção do fallback um estilo de tipo sem serifa, serifada, monospace etc.

Estilo de parágrafo

Para formatar o alinhamento de um parágrafo utilizamos a propriedade text-align:

text-align: left;
text-align: right;
text-align: center;
text-align: justify;

Para formatar a tabulação de um parágrafo (espaço antes da primeira linha) utilizamos a propriedade text-indent:

text-indent: 60px;

text-indent: 30pt;

Para controlar o espaçamento entre linhas, defina a altura da linha com a propriedade line-height:

line-height: 2rem;
line-height: 24pt;
line-height: 50px;

Na primeira opção, a linha terá o dobro de altura do valor raiz, na segunda, a linha terá 24pt de altura, se a fonte tiver 16pt por exemplo, terá 8pt de espaço vazio entre as linhas, na terceira opção a altura da linha será de 50px, incluindo o espaço ocupado pelo texto.

Fontes da web e de arquivo

Podemos ainda definir fontes externas, seja da internet, seja local. Para utilizar uma fonte da internet, basta adicionarmos ao início do arquivo CSS uma regra @import assim:

@import url('https://fonts.googleapis.com/css2?family=Bahianita&display=swap');
p { font-family: 'Bahianita', cursive; }

Neste exemplo importamos uma fonte da internet a partir da URL, a aplicamos aos parágrafos utilizando o nome da fonte e ainda definimos um fallback para uma fonte cursiva, semelhante à importada, que será utilizada caso a fonte on-line não esteja disponível ou não haja conexão com a internet.

Essa fonte foi retirada do site Google Fontes, confira:

fonts.google.com

Para utilizar um arquivo de fonte local, salvo em sua máquina utilize a regra @font-face, no início de seu arquivo CSS, assim:

@font-face {
    font-family: minhaFonte;
    src: url("Minhas Fontes/Playball-Regular.ttf");
}
p {
    font-family: minhaFonte;
}
