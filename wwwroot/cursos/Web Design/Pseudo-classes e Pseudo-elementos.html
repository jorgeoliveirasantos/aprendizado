Pseudo-classes e Pseudo-elementos

Pseudo-elementos

Pseudo-elementos são seletores específicos aplicados a determinada parte ou condição de um elemento HTML, por exemplo, ao criarmos um seletor CSS p::selection estamos indicando somente a parte do parágrafo que está selecionada pelo usuário. A sintaxe básica de um seletor de pseudo-elemento é a seguinte:

seletor::pseudoElemento { }

Selecionando a primeira letra de um texto para criar uma letra capitular:

<p>
No contexto da tipografia, uma letra capitular ou letra
capital é uma letra no início da obra, de um capítulo ou
de um parágrafo, de maior dimensão que o restante corpo
do texto.
Em manuscritos ou livros antigos, a letra capital é
muitas vezes profusamente decorada e chega a ocupar
várias linhas do corpo do texto.
</p>
<style>
p::first-letter {
  color: aqua;
  float: left;
  font-size: 60pt;
  line-height: 40pt;
  font-weight: bold;
  margin: 5px;
}
</style>

Selecionando a primeira linha de um texto:

<p>
No contexto da tipografia, uma letra capitular ou letra 
capital é uma letra no início da obra, de um capítulo ou
de um parágrafo, de maior dimensão que o restante corpo
do texto.
</p>
<style>
  p::first-line { background-color: #FF08; }
</style>

Inserir um elemento ao final de um contêiner:

<div>
  <p>www.jorgesouza.com</p>
</div>
<style>
div::after {
  content: "Saiba mais....";
}
</style>

Inserir conteúdo no início de um contêiner:

<div>
  <p>www.jorgesouza.com</p>
</div>
<style>
div::before {
  content: "Site:";
}
</style>

Selecionar os marcadores de uma lista:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<style>
li::marker {
  color: green;
}
</style>

Obter o texto selecionado:

<p>www.jorgesouza.com</p>
<style>
p::selection {
    color: fuchsia;
    background-color: yellow;
}
</style>

Selecionar o placeholder de um elemento input:

<input type="text" placeholder="Jorge">
<style>
input::placeholder {
  color: #0A07;
  font-weight: bold;
}
</style>

Pseudo-classes

Pseudo-classes funcionam de maneira semelhante aos pseudo-elementos, com uma pequena diferença na sintaxe, em vez de utilizarmos :: usamos :, assim:

seletor:pseudoClasse { }

Pseudo-classes são utilizadas para selecionar um elemento de acordo com seu estado, por exemplo:

p:hover { color: green; }

No exemplo acima, o estilo será aplicado somente quando o cursor do mouse estiver sobre o elemento. A pseudo-classe hover funciona como se chamássemos os eventos onmouseenter e onmouseleave no Javascript.

Manipulando o foco de um elemento:

<input type="text">
<style>
input:focus {
  color: green; background-color: black;
}
</style>

O elemento recebe o foco quando é clicado, e perde o foco quando clicamos fora dele. Nesse caso, o equivalente no Javascript seria o uso dos eventos onfocus e onblur.

Selecionando o elemento quando está ativo:

<button>Clique-me!</button>
<style>
button:active {
  color: green; background-color: black;
}
</style>

O elemento permanece ativo enquanto a ação do usuário ocorre. Neste exemplo, estará ativo enquanto o botão do mouse estiver pressionado sobre o elemento.

Selecionando um checkbox ou radio que está marcado:

<input type="checkbox">
<style>
input:checked { outline: yellow solid 2px; }
</style>

Neste exemplo, a aparência do checkbox será alterada quando ele for marcado.

Habilitado e desabilitado:

Podemos aplicar estilos diferentes para controles que estão habilitados ou desabilitados:

<input type="text">
<input type="text" disabled>
<style>
input:enabled { border: 2px solid green; }
input:disabled { border: 1px solid black; }
</style>

Válido e inválido:

Podemos estilizar um elemento de acordo com sua validação, por exemplo, se o usuário inseriu um valor correto ou incorreto:

<input type="email">
<style>
input:valid { border: 1px solid green; }
input:invalid { border: 2px solid red; }
</style>

Mínimo e máximo:

Podemos estilizar um elemento indicando que um valor inserido ultrapassou o limite definido:

<input type="number" min="5" max="15">
<style>
input:in-range { border: 1px solid green; }
input:out-of-range { border: 2px solid red; }
</style>

Para o usuário inserir um valor fora do limite estabelecido é preciso digitar, já que controles de nível param de incrementar e decrementar quando atingem o limite.

Opcional e obrigatório:

Podemos estilizar um elemento caso ele seja opcional ou obrigatório:

<input type="text">
<input type="text" required>
<style>
input:required { border: 2px solid blue; }
input:optional { border: 1px solid black; }
</style>

Índices e posições:

Podemos selecionar elementos de acordo com sua posição dentro de um contêiner, vejamos alguns exemplos:

Primeiro elemento filho:

<div>
    <p>Primeiro parágrafo</p>
    <p>Segundo parágrafo</p>
    <p>Terceiro parágrafo</p>
    <p>Quarto parágrafo</p>
    <p>Quinto parágrafo</p>
</div>
<style>
div p:first-child { border: 2px solid blue; }
</style>

No exemplo acima o primeiro parágrafo terá uma borda azul.

Último elemento filho:

div p:last-child {
  background-color: yellow;
  color: black;
}

Agora o último parágrafo terá o fundo amarelo e a cor preta.

Selecionando um elemento específico dentro do contêiner:

div p:nth-child(2) {
  background-color: yellow;
  color: black;
}

Neste exemplo o estilo será aplicado ao segundo elemento filho.

Elementos pares:

div p:nth-child(even) {
  background-color: yellow;
  color: black;
}

Elementos ímpares:

div p:nth-child(odd) {
  background-color: yellow;
  color: black;
}

Um a cada três elementos:

div p:nth-child(3n) {
  background-color: yellow;
  color: black;
}
