Estrutura de um documento HTML

Todo documento HTML pronto para publicação deve iniciar com uma tag <!DOCTYPE html> que define o tipo de documento a ser interpretado pelo navegador, em seguida abrimos o documento com uma tag <html>:

<!DOCTYPE html>
<html>
   <body>
      <h1>Título</h1>
      <h2>Subtítulo</h2>
      <p>Parágrafo</p>
      <!-- Comentário que não aparecerá no navegador -->
   </body>
</html>

Árvore

Um documento HTML assim como um documento XML é constituído de uma árvore de marcação que define a hierarquia de cada elemento, isso significa que um elemento pode ser pai ou filho de outro elemento, exemplo:

<elementopai>
  Conteúdo do elemento pai.
  <elementofilho>
     Conteúdo do elemento filho
  </elementofilho>
</elementopai>

Neste exemplo, um elemento está contido dentro do outro e herdará todas as suas propriedades, por exemplo, se o elemento pai define que o texto é vermelho, todo o conteúdo, tanto do elemento pai quanto do elemento filho serão vermelhos. No entanto, se o elemento filho definir que o texto será verde, nesse caso a propriedade a ser utiliza será a sua, e a herdada do elemento pai será ignorada.

Todo documento HTML é constituído de árvores de elementos que podem possuir a profundidade que for, o que significa que um elemento pode conter outro, que contém outro, que contém outro, e assim por diante.

Perceba no exemplo o espaçamento no início dos parágrafos (tabulação) que aumentam a cada novo nível, este padrão de espaçamento é chamado de “indentação” e é utilizado unicamente para facilitar a leitura do código, principalmente em arquivos grandes, mas não é obrigatório. Documentos HTML ignoram quebras de linha e espaços repetidos. Um documento pode muito bem ser escrito assim:

<html><body><h1>Meu título</h1><p>Meu parágrafo</p>
<button>Meu botão</button></body></html>

No entanto, perceba a dificuldade de manipulação que isso lhe traria. Outro detalhe é que o único elemento para o qual esta regra não vale é o elemento <pre> (pré-formatado), que exibe o texto exatamente como ele estiver no código.

Onde escrever documentos HTML

Apesar de existirem IDEs (Ambientes de Desenvolvimento Integrado) com bastante recursos para a criação e edição de código em diversas linguagens, documentos HTML podem ser escritos em qualquer editor, por exemplo no Bloco de Notas do Windows, bastando salvar o arquivo com a extensão .html para que possa ser interpretado por um navegador. Caso você queira utilizar um editor online uma boa opção é o Js Fiddle:

jsfiddle.net

Este editor já te fornece visualização do resultado e ainda permite adicionar CSS e Javascript. Existem ainda outros editores online que podem ser utilizados, mas caso você prefira utilizar uma IDE que forneça autopreenchimento de código, indentação automática, correção entre outros recursos, o mais utilizado é o Visual Studio Code que pode ser baixado em:

code.visualstudio.com

Ou utilizado on-line em:

vscode.dev

Comentários

Algumas partes de um código podem necessitar de explicações que apareçam apenas para quem lê o código fonte, isso pode ser feito adicionando linhas de código, assim:

<--! Algum comentário -->

Esta linha é ignorada pelo navegador, o que significa que não é renderizada e não aparecerá em seu site. Comentários podem conter instruções ou observações sobre o código para você mesmo ou para outra pessoa que ler o seu código.

Atributos

Marcas HTML podem possuir ainda atributos que definem propriedades aplicadas ao seu conteúdo.

Um atributo é inserido dentro da tag de abertura e tem este formato:

<minhatag atributo="valor"></minhatag>

ou:

atributo="propriedade1: valor1; propriedade2: valor2;"

Por exemplo, para inserir uma imagem utilizamos a tag <img> que necessita de um atributo que define o caminho da imagem que é o atributo src, o atributo src define o source, ou seja, a origem da imagem, assim:

<img src="pasta/subpasta/minhaimagem.jpg"/>

Um link necessita de um atributo href que define a referência, assim:

<a href="www.jorgesouza.com">
  Jorge Site
</a>

Para alterar o estilo, como a cor ou o alinhamento, por exemplo, utilizamos o atributo style, assim:

<p style="color: red; text-align: center;">
  Exemplo de parágrafo
</p>
