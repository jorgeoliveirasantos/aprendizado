Estilização do formulário

Como dito anteriormente, qualquer elemento HTML recebe estilos CSS padrão do navegador, em alguns casos isto pode ser suficiente para criarmos um formulário simples e obter alguma informação do usuário, no entanto ao desenvolver sites e aplicações Web de alto nível você precisará estilizar seus controles de forma a deixá-los, além de mais elegantes, também acessíveis, facilitando a compreensão das informações e do funcionamento de sua página. Neste tutorial aprenderemos a aplicar estilos personalizados a um formulário a fim de deixá-lo assim:

Começaremos obviamente com o HTML:

<form>
  <div>
    <fieldset>
      <legend>Dados de Login</legend>
      <label for="nome">Nome</label>
      <input type="text" id="nome"><br>
      <label for="sobrenome">Sobrenome</label>
      <input type="text" id="sobrenome"><br>
      <label for="email">E-mail</label>
      <input type="email" id="email"><br>
      <label for="senha">Senha</label>
      <input type="password" id="senha"><br>
    </fieldset>
  </div>
  <div>
    <fieldset>
      <legend>Dados Pessoais</legend>
      <label for="apelido">Apelido</label>
      <input type="text" id="apelido"><br>
      <label for="organizacao">Organização</label>
      <input type="text" id="organizacao"><br>
      <label for="cep">CEP</label>
      <input type="text" id="cep"><br>
      <label for="genero">Gênero</label>
      <select>
        <option>Masculino</option>
        <option>Feminino</option>
        <option>Outro</option>
      </select><br>
    </fieldset>
    <input type="submit" value="Registrar" id="btnRegistrar">
  </div>
</form>

Há muito pouco a se falar do código HTML já que você já deve possuir a esta altura a teoria e a prática básicos para desenvolvê-lo, nele contém um pouco do que aprendemos como labels, campos de texto e seleção múltipla. Aplicaremos os estilos CSS diretamente aos elementos HTML, se possuir mais de um formulário na mesma página e desejar aplicar estilos diferentes, você terá de utilizar ids ou classes.

Não se esqueça de definir a id do botão!

Primeiro criamos o elemento style, você pode utilizar um CSS externo se preferir:

<style></style>

Dentro das tags começaremos a criar nossos estilos. Primeiro estilizamos o formulário como um todo, definindo cor do conteúdo, do fundo, a largura e as margens:

 form {
  width: fit-content;
  margin: auto;
  padding: 25px;
  background-color: black;
  color: aquamarine;
}

Depois o fieldset e a legenda:

fieldset {
  border: white 1px solid;
  margin: 10px;
}
legend {
  color: white;
}

Enfim, os controles. Primeiro as labels:

label {
  display: inline-block;
  width: 120px;
  height: 20px;
  margin: 4px;
  padding: 4px;
}

Depois os campos de input:

input {
  display: inline-block;
  width: 150px;
  height: 20px;
  margin: 4px;
  padding: 4px;
  border: black 1px solid;
  border-radius: 14px;
}

O elemento select e seus subelementos option devem ser estilizados separadamente:

select {
  display: inline-block;
  width: 160px;
  height: 30px;
  margin: 4px;
  padding: 4px;
  border: black 1px solid;
  border-radius: 14px;
}
option {
  background-color: black;
  color: aquamarine;
  height: 50px;
}

Agora vamos aplicar um estilo bacana ao botão submit! Quando estilizamos um botão ele perde as características default, então devemos utilizar a pseudo-classe hover para estilizar também a aparência quando o usuário passa o mouse sobre o botão:

#btnRegistrar {
  background-color: darkslategray;
  color: white;
  width: 100%;
  height: 40px;
  border-radius: 20px;
  margin: none;
  font-size: large;
  font-weight: bold;
}
#btnRegistrar:hover {
  background-color: white;
  color: indigo;
  cursor: pointer;
}
