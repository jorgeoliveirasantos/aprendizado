Atributos dos controles

Cada controle de um formulário possui atributos próprios além dos atributos globais que são aplicáveis a ele, a partir de agora daremos uma olhada geral nos atributos mais importantes que você precisará utilizar ao estruturar seu formulário.

Value e placeholder

Os atributos value e placeholder são utilizados para predefinir um valor padrão para os campos, o atributo value preenche o campo com um valor editável, enquanto o atributo placeholder preenche com uma sugestão que desaparece ao começarmos a inserir dados no campo.

<input type="text" placeholder="exemplo placeholder">
<input type="text" value="exemplo value">

Todos os controles de formulário podem receber o atributo value.

O atributo placeholder é aplicável apenas a controles de texto: text, email, tel, url, search, password e number.

Size, height e width

Os controles recebem um estilo CSS padrão do navegador ao serem carregados, aplicando propriedades que não definimos explicitamente, essas propriedades que dirão a aparência e o tamanho dos campos, mas podemos alterá-las. O atributo size é aplicável apenas a campos de texto e possui a mesma função que o atributo width, ou seja, define a largura. Já os atributos height e width são aplicáveis apenas ao controle <image> que serve para criar um botão submit utilizando uma imagem.

<input type="tel" size="50px">
<input type="image" src="minhaImagem.jpg" width="30px" height="30px">

Atenção! Criar um elemento input com o tipo image no formulário, não é a mesma coisa que criar um elemento img, este tipo de input é um botão e quando clicado enviará o formulário.

Minlength e maxlength

Os controles que aceitam texto podem definir uma quantidade máxima de caracteres aceita, isso facilita a garantia de que o usuário inserirá um valor válido e também garante a segurança do formulário, impedindo que o usuário insira código malicioso (geralmente longo) nos campos de texto.

O atributo minlegth perderá efeito se o formulário possuir um atributo novalidate. Já o atributo maxlength não permite inserir caracteres além do limite. 

<input type="text" maxlength="20">
<input type="password" minlength="5">

Min, max e step

Os atributos min, max e step são utilizados em controles de data, hora e nível, enquanto min e max definem o valor mínimo e máximo, o atributo step define a escala de acréscimo e decréscimo (2 em 2, 3 em 3, etc.).

<input type="range" min="10" max="20" step="2">

Required

O atributo required define um campo como obrigatório, o que significa que o formulário não poderá ser submetido se o campo for deixado em branco:

<input type="text" required>

O atributo required perderá efeito se o formulário for marcado como novalidate.

Todos os controles podem receber o atributo required exceto os controles color e range, ou controles marcados como hidden.
