Tabelas

Tabelas são componentes HTML com quatro níveis, a tabela em si, o corpo e o cabeçalho, as linhas e as células que juntas formam as colunas, uma estrutura de tabela é semelhante a esta:

<table>
  <!-- Cabeçalho -->
  <thead>
    <th>Título 1</th>
    <th>Título 2</th>
    <th>Título 3</th>
  </thead>
  <!-- Corpo -->
  <tbody>
    <tr>
      <td>Célula 1</td>
      <td>Célula 2</td>
      <td>Célula 3</td>
    </tr>
  </tbody>
</table>

A tag <thead> define o cabeçalho, onde estarão os títulos de cada coluna, estes títulos são definidos nas tags <th>. A tag <tbody> define o corpo da tabela, dentro do corpo inserimos as linhas <tr>, dentro de cada linha podemos inserir uma quantidade de células <td>, geralmente esta quantidade coincide com a quantidade de títulos inseridos no cabeçalho. Quando a tabela está montada, cada célula ficará abaixo de um título correspondente, formando assim as colunas.

Span

O span é utilizado para fazer com que uma célula ocupe duas ou mais linhas na vertical, ou duas ou mais colunas na horizontal, temos dois atributos de span que podemos aplicar a uma célula, colspan, o span de coluna, e rowspan, o span de linha. Vejamos como funciona:

<table>
  <thead>
    <th>Título 1</th>
    <th>Título 2</th>
  </thead>
  <tbody>
    <tr>
      <td colspan="2">Célula</td>
    </tr>
  </tbody>
</table>

Neste exemplo, a célula tem um colspan definido como 2, o que significa que ela ocupa sua própria coluna e a próxima coluna. Para que não haja um estouro da tabela é necessário que a próxima coluna não contenha uma célula, caso contrário o HTML criará mais uma coluna na tabela para acomodar todos os elementos. O mesmo é válido para o span de linha, vejamos um exemplo de tabela onde usamos todos os dois atributos:

<table>
  <thead>
    <th>Título 1</th>
    <th>Título 2</th>
    <th>Título 3</th>
  </thead>
  <tbody>
    <tr>
      <td colspan="2">Célula 1</td>
      <td rowspan="2">Célula 2</td>
    </tr>
    <tr>
      <td colspan="2">Célula 1</td>
    </tr>
  </tbody>
</table>

Veja no exemplo acima, que temos três colunas definidas pelos três títulos, mas na primeira linha criamos apenas duas células, pois a primeira célula ocupará duas colunas. A segunda linha possui apenas uma célula, isso porque a célula 1 na segunda linha ocupa as duas primeiras colunas e a célula 1 da primeira linha ocupará também a segunda linha.

Estilizando linhas e colunas da tabela

Para unir as linhas e colunas de uma tabela, ou seja, remover o espaço em branco que as separa, devemos colapsar as bordas com a propriedade border-collapse:

table { border-collapse: collapse; }

Então podemos estilizar a tabela de diversas maneiras. Utilizando o que já sabemos de CSS, você pode estilizar toda a tabela, títulos ou células individualmente, mas veremos como estilizar linhas e colunas inteiras com a pseudo-classe nth-child():

Supõe-se que você estudou o primeiro capítulo HTML Básico que possui conceitos fundamentais dos quais necessitaremos a partir de agora.

Estilizando a terceira linha da tabela:

table {
   border-collapse: collapse; background-color: aqua;
}
tr:nth-child(3) {
   background-color: lightgreen;
}

Neste exemplo, a terceira linha terá uma cor diferente do restante da tabela.

Estilizando uma em cada três linhas:

tr:nth-child(3n) {
   background-color: lightgreen;
}

Estilizando a segunda coluna:

td:nth-child(2) {
   background-color: lightgreen;
}

Estilizando a segunda coluna incluindo o título:

th:nth-child(2), td:nth-child(2) {
	background-color: lightgreen;
}

Estilizando colunas ímpares, incluindo o título:

th:nth-child(odd), td:nth-child(odd) {
	background-color: lightgreen;
}

Estilizando linhas pares:

tr:nth-child(even) {
	background-color: lightgreen;
}

Mesclando cores na tabela

Ao aplicar um estilo a linhas e colunas, as cores não serão mescladas, mas sobrepostas, criando um resultado que provavelmente não seria o desejado, o que é feito nesses casos é utilizar cores com um grau de transparência fazendo uso da escala RGBA, feita no modelo rgba(red, green, blue, alpha) que estudaremos no momento certo, mas que funciona assim:

rgba(255, 255, 255, 1);

Este exemplo cria uma cor branca totalmente opaca (sem transparência), pois adicionamos vermelho, verde e azul ao máximo, e definimos a opacidade ao máximo, que é 1, veja este outro exemplo:

rgba(255, 128, 128, 0.5);

Neste exemplo temos um vermelho claro, pois possui um valor máximo de vermelho, e metade dos valores de azul e verde, e está definido com 50% de transparência, 0.5 do canal alfa.

rgba(128, 0, 0, 1);

Já este exemplo produz um vermelho-escuro sem transparência. Agora que temos um pouco da teoria, vamos criar um CSS básico para fazer uma lista de alunos, crie uma tabela com 5 linhas e quatro colunas, e preencha-a conforme o exemplo da tabela a seguir:
