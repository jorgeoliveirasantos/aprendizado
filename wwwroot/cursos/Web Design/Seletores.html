Seletores

São os seletores que definem quem e em que condições receberá um conjunto de propriedades de estilo, temos desde seletores universais, como por exemplo o caractere curinga * que aplica seu estilo a todos os elementos de forma arbitrária, a seletores complexos que aplicam seu estilo a elementos filho de outros elementos ou que contenham determinado atributo, ou sobre determinada condição, como quando ele é precedido por outro elemento, ou somente quando o ponteiro do mouse está sobre ele.

Estilos universais podem ser aplicados com um caractere curinga:

<body>
 <h1>Meu título</h1>
 <p>Meu parágrafo</p>
</body>
<style>
* {
    background-color: black;
    color: red;
    font-family: sans-serif;
}
</style>

Ou aplicando diretamente ao corpo do documento:

body {
    background-color: black;
    color: red;
    font-family: sans-serif;
}

Diferentemente do caractere curinga, aplicar um estilo ao <body>, possui um peso maior de decisão e pode substituir outros estilos aplicados posteriormente. No entanto, a aplicação de estilo universal a todo o documento é desaconselhável, pois pode gerar comportamentos indesejados, com um pouco mais de código você consegue definir estilos para vários elementos.

Seletor de id e classe

Seletores de id iniciam com uma cerquilha # e seletores de classe com um ponto ., o seletor de id aplicará as propriedades ao elemento específico que possui aquela id, se a id for repetida, o que não deve ocorrer, somente o primeiro elemento que a possua receberá o estilo. Já o seletor de classe aplicará suas propriedades a todos os elementos que compartilham aquela classe.

<body>
 <p id="minha-id" class="minha-classe">Meu parágrafo</p>
</body>
<style>
/* Seletor de classe */
.minha-classe {
    color: red;
}
/* Seletor de id */
#minha-id {
    background-color: green;
}
</style>

Seletores de tipo

Os seletores de tipo selecionam elementos HTML como <p>, <h1>, <span> etc, e aplicarão suas propriedades a todos os elementos do documento que pertençam àquele tipo:

h1 {    color: red; }


b {  color: blue; }

Seletores de atributo

Seletores de atributo aplicam seu estilo a um elemento que contenha determinado atributo:

<p lang="en-US">My paragraph</p>
<style>
  [lang] { font-style: italic; }
</style>

Neste exemplo qualquer elemento que possuir o atributo lang, receberá o estilo, não importa o valor que o atributo contenha. Podemos utilizar de diferentes maneiras.

Elementos <p> que contenham o atributo lang:

p[lang] { font-style: italic; }

Elementos <p> que contenham o atributo lang="en-US":

p[lang="en-US"] { font-style: italic; }

Seleção múltipla

Um estilo pode ser aplicado a vários elementos separando-os por vírgula:

p, h1, div, .minhaClasse, #minhaId { color: yellow; }

Aqui, todos os elementos separados por vírgula no seletor terão a cor amarela.

Elemento filho

Para selecionar um elemento filho de outro elemento basta separá-los por um espaço, no exemplo a seguir selecionamos o elemento p que está contido dentro de uma div:

div p { color: yellow; }

O espaço seleciona um elemento dentro de outro elemento em qualquer nível, isso quer dizer que não precisa ser um descendente direto.

Elemento irmão

Podemos ainda selecionar um elemento que seja irmão de outro elemento utilizando o sinal de +, ou seja, que esteja no mesmo nível hierárquico, ou compartilhando o mesmo elemento pai. No exemplo a seguir, estamos selecionando o elemento p que venha logo após um elemento h1:

h1 + p { color: yellow; }

Já neste exemplo, estamos selecionando o próximo elemento p, que venha logo após um elemento h1, estando ambos dentro de um elemento div, em qualquer nível:

div h1 + p { color: yellow; }

Descendente direto

	Para evitar que um estilo se propague a toda a hierarquia de descendentes utilizamos o seletor >, no exemplo abaixo estamos selecionando apenas o elemento p que seja descendente direto de uma div:

div > p { color: yellow; }



	E neste exemplo estamos selecionando o próximo elemento p que venha logo após um elemento h1, devendo ambos estarem no primeiro nível dentro de uma div:


div > h1 + p { color: yellow; }



Seleção composta


	Seletores compostos permitem definir duas ou mais condições simultâneas que devem ser respeitadas para que o estilo seja aplicado a um elemento. Neste exemplo, o seletor apenas selecionará o elemento p que contenha um atributo id definido como minhaId:


<p id="minhaId">Meu parágrafo</p>
<style>
	p#minhaId { color: yellow; }
</style>



	Neste exemplo, somente os atributos p que contenham o atributo class definido como minhaClasse serão selecionados:


p.minhaClasse { color: red; }
	Podemos combinar dois atributos, ou até mesmo, outras regras de seleção de CSS, neste exemplo, somente o elemento que contiver a id e a classe especificadas será selecionado:

.minhaClasse#minhaId { color: red; }
