Box Sizing

A propriedade box-sizing define como as dimensões da caixa de um elemento são calculadas. Todos os elementos HTML são caixas, desde um parágrafo comum até uma tabela ou uma imagem. Cada elemento possui conteúdo, padding, borda e margem, exceto o conteúdo, as outras propriedades se não forem definidas terão um valor 0. Um parágrafo <p> sem um estilo CSS definido terá um tamanho definido pela fonte, padding 0, border 0 e margin 0. Portanto o tamanho da caixa é definido pelo tamanho do conteúdo, ou seja, pelo texto.

Considere o seguinte estilo aplicado ao parágrafo:

p{
   background-color: brown;
   width: 200px;
   padding: 20px;
   border: yellow 5px solid;
}

No exemplo acima a largura total da caixa à qual pertence o parágrafo é de 250px, sendo 200 de largura do conteúdo mais 40 de padding, 20 de cada lado, mais 10 de borda, 5 de cada lado. Portanto a propriedade width é aplicada ao conteúdo ignorando o padding e a borda, e os valores sobressalentes são somados. Isso significa que o tamanho do elemento será maior do que o que definimos.

Agora considere o seguinte estilo:

p{
   background-color: brown;
   width: 200px;
   padding: 20px;
   border: yellow 5px solid;
   box-sizing: border-box;
}

Já neste exemplo a caixa será menor que no anterior, mesmo que a propriedade width seja a mesma, isso porque agora o cálculo é feito pela caixa de borda e não de conteúdo, o que significa que o width que definimos é aplicado à borda.
