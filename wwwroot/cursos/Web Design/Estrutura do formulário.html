Estrutura do formulário

A estrutura de um formulário é toda criada entre as tags <form>, geralmente teremos um par de <label> e controle de entrada, o label é a descrição do controle de entrada, por exemplo:

<form>
  <label>Nome:</label>
  <input type="text">
</form>

Aqui temos apenas um par, ou seja, apenas um controle, este formulário ficaria assim:

Vejamos um exemplo de formulário básico com dois controles de entrada de texto e um botão de envio:

<form>
  <label>Nome:</label>
  <input type="text">
  <label>Sobrenome:</label>
  <input type="text">
  <input type="submit" value="Enviar">
</form>

O resultado deste formulário é:

Neste exemplo temos um elemento form que serve de contêiner para controles de formulário, utilizamos apenas dois tipos de controle, label e input. O controle label é utilizado para criar a etiqueta que identifica o controle de entrada, sendo assim, cada controle de um formulário é criado em par com um label.

Note que o tipo de controle que utilizaremos no input não é definido pela tag e sim pelo atributo type. Temos então dois tipos de controles, de texto e um botão de enviar, diferente do elemento button, o texto do botão submit de um formulário é definido pelo atributo value, no entanto, na maioria dos casos em vez de utilizar um input submit, utiliza-se um button comum, veremos isso adiante.

O controle label no exemplo anterior não estava vinculado ao seu par, essa vinculação é feita com um atributo for, assim:

<label for="nome">Nome:</label>
<input id="nome" type="text">
<label for="sobrenome">Sobrenome:</label>
<input id="sobrenome" type="text">

Definimos um atributo de id para cada input, e a passamos no atributo for das labels, dessa maneira, clicando na label, sua caixa de texto correspondente recebe o foco automaticamente. As labels podem ainda selecionar ou tirar a seleção de itens:

<label for="nome">Nome:</label>
<input id="nome" type="checkbox">
<label for="sobrenome">Sobrenome:</label>
<input id="sobrenome" type="checkbox">

Agora você pode marcar ou desmarcar as caixas de seleção não somente clicando nelas, mas também em suas etiquetas.
