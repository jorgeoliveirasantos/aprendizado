Flexbox

A propriedade flexbox surgiu para facilitar a criação de layouts flexíveis e responsivos diminuindo em grande parte o trabalho de organizar, dimensionar e posicionar itens dentro de um contêiner de modo adequado. Vamos supor que você possua o seguinte HTML:

<div class="conteiner">
    <img src="1.png">
    <img src="2.png">
    <img src="3.png">
    <img src="4.png">
    <img src="5.png">
    <img src="6.png">
    <img src="7.png">
    <img src="8.png">
    <img src="9.png">
    <img src="10.png">
</div>

Supondo que cada imagem possua tamanhos e proporções diferentes, elas serão dispostas conforme aparecem respeitando o espaço e a margem da imagem anterior, fazendo com que a disposição seja totalmente arbitrária.

Agora experimente adicionar o seguinte estilo:

.conteiner {
    display: flex;
    flex-wrap: wrap;
    flex-direction: row;


}


.conteiner img {


    padding: 10px;


    margin: 10px;


    background: yellow;


}

Conseguiu notar as diferenças? Agora não temos mais lacunas entre as imagens e além disso a disposição é feita em linhas onde todas as imagens possuem a mesma altura. Isso se deve por causa dessas propriedades:

display: flex: Essa propriedade cria a caixa flex que organiza os elementos dentro do contêiner em linhas ou colunas ajustando seus tamanhos.

flex-wrap: wrap: Essa propriedade faz com que os elementos dentro do contêiner pulem de linha ao chegar ao final da tela.

flex-direction: row: Essa propriedade faz com que os elementos sejam dispostos em linhas e não em colunas.

Flex-wrap

Experimente substituir a propriedade flex-wrap da seguinte maneira:

flex-wrap: wrap-reverse;

Esse valor faz com que os elementos sejam ordenados inversamente, nesse caso, iniciando na imagem 10.

Agora experimente alterar para:

flex-wrap: nowrap;

Esse é o valor padrão da propriedade flex-wrap, ela produz uma única linha contínua com todos os elementos e a depender da quantidade gera um overflow criando uma barra de rolagem na horizontal, estudaremos sobre overflow logo adiante.

Flex-direction

Vejamos agora a finalidade da propriedade flex-direction, para tal, altere essa propriedade para:

flex-direction: column;

Agora os itens serão dispostos um abaixo do outro, ou seja, numa única coluna.

Experimente redimensionar o tamanho da janela do navegador!

Também podemos inverter a ordem dos itens da seguinte maneira:

flex-direction: column-reverse;

Agora, vamos fazer algo ainda mais avançado, altere o estilo e adicione mais uma linha nas propriedades de imagem, da seguinte maneira:

.conteiner {
    display: flex;
    flex-wrap: wrap;
    flex-direction: row;
}
.conteiner img {
  ...
  min-width: 100px;
}

Dessa forma estamos definindo um valor mínimo de largura para os itens dentro do contêiner e definindo uma quebra de linha ao chegar ao final da tela, dessa maneira os itens são redimensionados conforme o tamanho da tela diminui, e ao exceder a largura mínima pulam para a próxima linha.

Isso funciona desde redimensionar a janela do navegador até acessar o site em diferentes tamanhos de tela.

Flex-basis

A propriedade flex-basis é utilizada para definir o tamanho mínimo dos itens em cada linha, seja de largura ou de altura, e funciona de maneira semelhante ao exemplo anterior:

.conteiner {
    display: flex;
    flex-wrap: wrap;
    flex-direction: row;
}
.conteiner img {
    padding: 10px;
    margin: 10px;
    background: yellow;
    flex-basis: 200px;
}

Dessa maneira, se o último item da linha exceder o limite mínimo de 200px e não couber passa para a próxima.

Flex-grow

A propriedade flex-grow permite definir se aceitamos ou não espaços vazios ao final das linhas, o valor padrão, que é o que vinha sendo utilizado é 0, podemos definir um fator de crescimento entre 0 e 1, sendo que um não permitirá espaço vazio ao final da linha:

.conteiner img {
    ...
    flex-grow: 1;
}

Veja o resultado:

Redimensione a janela do navegador e veja o comportamento da caixa flex.

A propriedade flex-grow pode ser utilizada ainda para dimensionar os elementos de maneira independente, por exemplo:

<div class="conteiner">
    <p class="p1">Item 1</p>
    <p class="p1">Item 2</p>
    <p class="p1">Item 003</p>
    <p class="p2">Item 00004</p>
    <p class="p2">Item 00005</p>
</div>
<style>
.conteiner { 
    display: flex;
    flex-wrap: wrap;
}
.conteiner p {
    padding: 50px;
    background-color: brown;
    color: yellow;
    border: cyan 10px solid;
    font-size: xx-large;
}
.conteiner .p1 {  flex-grow: 0.5;  }
.conteiner .p2 {  flex-grow: 1;  }
</style>

No exemplo acima, além do fato de que não haverá espaços em branco ao final da linha, os elementos p1 terão uma largura inicial definida como metade da largura dos elementos p2. No entanto, todos os elementos são redimensionados com base na largura inicial para preencher a linha, de modo a não deixar espaços vazios.

Flex-shrink

A propriedade flex-shrink é muito semelhante à flex-grow, com a diferença de que ela redimensiona os elementos sem preencher a linha, isto é, deixando espaços vazios ao final.

<div class="conteiner">
    <p class="p1">Item 1</p>
    <p class="p1">Item 2</p>
    <p class="p1">Item 003</p>
    <p class="p2">Item 00004</p>
    <p class="p2">Item 00005</p>
</div>
<style>
.conteiner { 
    display: flex;
    flex-wrap: wrap;
}
.conteiner p {
    padding: 50px;
    background-color: brown;
    color: yellow;
    border: cyan 10px solid;
    font-size: xx-large;
}
.conteiner .p1 {  flex-shrink: 0.5;  }
.conteiner .p2 {  flex-shrink: 1;  }
</style>

Alinhamento

Podemos combinar propriedades para obter um controle ainda maior sobre o comportamento de caixas flex. Com a propriedade align-items podemos definir como os elementos serão alinhados na vertical dentro da caixa flex. Considere o seguinte código:

<div class="conteiner">
    <p>Item 1</p>
    <p>Item 2</p>
    <p>Item 003</p>
    <p>Uma linha de texto longa</p>
</div>
<style>
.conteiner {
    background-color: darkblue;
    display: flex;
    flex-wrap: wrap;
    align-items: center;
}
.conteiner p {  flex-basis: 100px;  }
</style>

Neste exemplo, os itens serão alinhados no centro da linha da caixa flex. Vejamos outros exemplos:

Alinhados ao topo:

align-items: flex-start;

Alinhados à base:

align-items: flex-end;

Alinhados ao topo e à base (esticados pela margem):

align-items: stretch;

Justificação

A propriedade justify-content além de justificar os itens verticalmente permite alinhá-los horizontalmente na linha do contêiner.

Alinhados ao topo e à base (esticados pela margem), e dispostos no início da linha do contêiner:

justify-content: baseline;

Itens esticados ao topo e a base e alinhados no centro da linha do contêiner:

justify-content: center;

Alinhados ao início da linha do contêiner:

justify-content: start;

Alinhados ao fim da linha do contêiner:

justify-content: end;

Alinhados à esquerda da linha do contêiner:

justify-content: left;

Alinhados à direita da linha do contêiner:

justify-content: right;

Itens distribuídos pela linha com seu tamanho inicial mantendo espaços vazios ao redor deles de modo a preencher a linha:

justify-content: space-around;

Itens distribuídos pela linha com seu tamanho inicial mantendo espaços vazios entre eles de modo a preencher a linha (não mantém espaços no fim e no início da linha):

justify-content: space-between;

Itens distribuídos pela linha com seu tamanho inicial mantendo espaços vazios iguais ao redor deles (o espaço vazio entre os itens é colapsado para coincidir com o tamanho dos espaços no início e no fim):

justify-content: space-evenly;
