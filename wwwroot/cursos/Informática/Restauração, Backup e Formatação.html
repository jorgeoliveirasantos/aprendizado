Restauração, Backup e Formatação

Em muitos casos apenas uma reinstalação completa do Windows pode solucionar um problema, por isso é essencial o usuário conhecer o processo de backup, formatação e instalação do sistema operacional.

Esse processo pode ser usado até mesmo para atualizar a versão do sistema ou migrar de um computador para o outro.

Imagem do Windows

A instalação do Windows requer uma imagem do sistema operacional, é esta imagem que instalará todo o novo sistema quando o anterior for excluído.

Durante a instalação o que rodará no computador é a imagem, isso possibilita que o sistema operacional anterior possa ser desinstalado ainda com o computador ligado.

Antes de iniciar um processo como este verifique se seu computador suporta o sistema que você quer instalar.

Você pode baixar uma imagem do Windows diretamente no site da Microsoft:

https://www.microsoft.com/pt-br/software-download/windows10

Mídia de Instalação

No site da Microsoft você tem a opção de baixar uma ferramenta que criará sua mídia de instalação, lembre-se que durante o processo de instalação, todo o sistema operacional será removido, por isso você precisará de uma mídia externa onde esta imagem será instalada. Caso não tenha essa possibilidade você ainda tem a opção de baixar a imagem e salvá-la numa segunda partição do disco que não seja a que comporta o sistema operacional.

Este processo pode ser simplificado deixando tudo por conta do próprio sistema operacional utilizando ferramentas de reinstalação existentes no próprio Windows.

Instalação do Windows

Durante a instalação do sistema operacional você terá três opções:

Manter arquivos configurações e programas:

Essa opção apenas restaurará a imagem do sistema operacional existente no computador.

Manter apenas arquivos:

Essa opção restaurará o sistema operacional por completo excluindo os programas instalados.

Não manter nada:

Essa opção excluirá tudo no disco ou partição em que o Windows está instalado, incluindo os arquivos pessoais e reinstalará o sistema operacional. Nada acontece com partições secundárias do disco.

Backup

Quando precisamos reinstalar completamente o sistema sem manter nada, é necessário fazer backup dos arquivos, pois os mesmos serão excluídos durante o processo de formatação. O backup consiste em copiar os arquivos existentes (ou somente os necessários), para outro local que não seja afetado pela formatação, pode ser para outro computador na rede, para um serviço de nuvem, para uma mídia externa como um pendrive, SSD ou HD, ou até mesmo numa segunda partição que não seja afetada pela formatação.

Backup em armazenamento externo:

Você pode copiar seus arquivos para um HD ou SSD externa, pen-drive ou outro dispositivo de armazenamento externo, a cópia pode ser feita manualmente pelo Explorador de Arquivos, ou configurada para ser feita com frequência pelo sistema operacional por meio das opções de backup.

Em nuvem:

Caso você possua uma assinatura de armazenamento de arquivos em nuvem com capacidade de guardar todos os seus arquivos, você pode fazer o upload deles caso o programa de sincronização já não o tenha feito. Nesse caso, após a formatação será preciso instalar os drivers de rede e conectar o computador à internet para recuperá-los.

Numa segunda partição do disco:

É possível ainda criar uma segunda partição de disco e mover seus arquivos para ela. Nesse caso, é necessário atenção na hora da instalação do Windows para escolher a partição em que o Windows está instalado, e não a que contém seus arquivos, para não os perder. É ideal que ao particionar deixe as duas partições com tamanhos diferentes para que possa identificá-las no momento da formatação. Para particionar o disco clique no menu iniciar e digite “Criar e Formatar Partições do Disco Rígido”.

Todo cuidado é pouco na hora de formatar uma partição para não perder os arquivos pessoais armazenados no computador.

Utilizando as opções de backup para salvar os arquivos pessoais do computador

O Windows conta com uma opção de backup automático em mídia externa que pode ser definido nas configurações do Windows em: Configurações → Atualização e Segurança → Backup.

Note que você pode efetuar este mesmo processo para agendar backups frequentes que serão feitos em intervalos entre a cada 10 minutos e diariamente. Você pode utilizar essa opção para efetuar um backup em mídia externa antes de formatar o PC.

Para tal, siga o processo abaixo:

1. Abras o aplicativo Configurações.

2. Clique em Atualização e Segurança.

3. Clique em Backup.

4. Clique em Adicionar unidade.

5. Escolha a unidade em que deseja fazer o backup (a mídia externa deve estar conectada ao computador).

6. Assim que a opção de backup for ativada, clique em Mais opções.

7. Em: Fazer backup destas pastas, clique em Adicione uma pasta, e adicione as pastas que você deseja salvar.

8. Na lista de pastas adicionadas, se houver alguma que você não deseja efetuar backup, basta clicar sobre ela e selecionar Remover.

9. Clique em Fazer backup agora.

Para cancelar backups agendados, siga até esta página, role até o final, e escolha “Parar de usar esta unidade”. Para restaurar um backup feito anteriormente, siga até esta página, role até o final, e escolha “Restaurar arquivos de um backup atual”.

Utilizando as opções de recuperação para formatar o computador

Para utilizar as opções de recuperação do Windows para executar a formatação, primeiro efetue um backup de seus arquivos, em seguida siga estas etapas:

1. Abras o aplicativo Configurações.

2. Clique em Atualização e Segurança.

3. Clique em Recuperação.

4. Clique em Começar agora.

5. Escolha Manter meus arquivos, caso deseja apenas remover os aplicativos, configurações e programas.

6. Escolha Remover tudo, caso prefira uma reinstalação limpa e completa.

7. Clique em Avançar.

8. Clique em Restaurar.

Se o motivo da formatação for a remoção de um vírus que estava causando a anormalidade no computador, o ideal é não manter nada. Assim que a formatação for feita, se for copiar arquivos de um backup, uma verificação com o antivírus deve ser feita neles.
