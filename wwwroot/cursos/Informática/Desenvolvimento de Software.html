Desenvolvimento de Software

O setor de desenvolvimento de software é um setor promissor para quem deseja entrar no mercado de trabalho. Há muito que eu poderia falar aqui, mas vou passar apenas o básico para quem se interessar em tornar-se um programador de aplicativos:

1. Decida em que área deseja trabalhar, em desenvolvimento de aplicativos para Desktops, desenvolvimento Móvel (Smartphones) ou IoT – Internet das Coisas (para tvs, veículos etc.).

2. Escolha uma linguagem de programação e estude o máximo que puder sobre ela, principalmente sua sintaxe básica.

3. Escolha um framework para trabalhar com a linguagem escolhida e estude o máximo que puder sobre ele.

Por vezes, você pode pensar que enlouquecerá ao se deparar com a quantidade imensa de informações para um único assunto que pesquisar, então para facilitar vou te dar duas diretrizes básicas:

1. Faça um curso superior de Engenharia de Software ou de Ciência da Computação.

2. Caso ainda queira fazer tudo de forma autodidática uma dica que dou para deixar tudo fácil é aprender inicialmente linguagens mais simples como Javascript, C# ou Python e utilizar compiladores on-line para executar o código até ir pegando o jeito.

3. Outra opção é aprender o desenvolvimento para internet, de sites e aplicações web, para tal você deverá estuda HTML, CSS e Javascript.

Caso queira deixar todas as decisões iniciais por conta de alguém que já passou por este processo e entrar numa metodologia que te levará do zero ao desenvolvimento profissional de softwares completos para as diversas plataformas e dispositivos, faça o Curso Completo de Programação no site www.jorgesouza.com.br

Existe um tipo simples de software fácil de desenvolver que é chamado de Software de Linha de Comando, e tem esse nome pois opera apenas com o código de fundo, sem uma interface de usuário. Vamos criar um agora?

O software que criaremos agora fará uma limpeza nos arquivos temporários do computador para aumentar sua velocidade e mantê-lo saudável. Para isso utilizaremos uma linguagem de programação de baixo nível chamada Bash que é a linguagem operada pelo Prompt de Comando. Abra o Bloco de Notas e escreva o código a seguir:

DEL /F /S /Q %SystemRoot%\TEMP\*
DEL /F /S /Q %SystemRoot%\Prefetch\*
DEL /F /S /Q %SystemRoot%\SoftwareDistribution\Download\*
DEL /F /S /Q %USERPROFILE%\AppData\Local\Temp\*
pause

Vamos entender o código do nosso programa:

DEL – É o comando utilizado para excluir arquivos.

Em seguida temos os parâmetros do comando DEL, que são /F /S /Q, respectivamente:

/F – É usado para forçar a exclusão de arquivos somente leitura.

/Q – Modo silencioso, exclui os arquivos sem solicitar confirmação do usuário.

/S – Estabelece que o sejam excluídos os arquivos da pasta atual e de todas as subpastas.

%SystemRoot% - Diretório raiz do sistema (C:\Windows).

Em seguida passamos os diretórios de onde queremos excluir os arquivos, na primeira linha foi a pasta \Temp, que guarda os arquivos temporários, na segunda linha foi a pasta \Prefetch, na terceira a pasta \SoftwareDistribuition e na quarta a pasta \Temp do usuário.

\* - É o caractere curinga utilizado para definir que todos os arquivos no diretório serão excluídos, sem ele o sistema tenta excluir a pasta e não seu conteúdo.

pause – este comando impede que a janela do prompt feche abruptamente após a conclusão, pausando para que o usuário possa ver tudo o que foi feito.

Agora, salve o arquivo de texto na sua Área de Trabalho com o nome “Limpeza.bat”, sem as aspas. O formato .bat é um formato de arquivo de comandos em lote, utilizado para executar comandos do prompt. Abra a Área de Trabalho, localize o arquivo Limpeza.bat, clique com o botão direito do mouse sobre ele e selecione “Executar como Administrador”, uma caixa de diálogo solicitará sua confirmação, clique em Sim.

Pronto! Você já tem um programa que faz a exclusão de arquivos temporários que se acumulam com o tempo deixando o computador lento, e o melhor de tudo, foi você mesmo o programou!
